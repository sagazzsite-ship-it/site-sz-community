📜 Regras da Comunidade

COMMUNIDADE SZ
├─ GERAL
│  ├─ Respeito é obrigatório entre todos os membros
│  ├─ Racismo, preconceito, homofobia ou discurso de ódio = punição
│  ├─ Brigas, tretas e provocações não são permitidas
│  ├─ Linguagem ofensiva excessiva será punida
│  └─ Permanecer no servidor significa aceitar todas as regras
├─ TEXTOS
│  ├─ Use cada canal para sua finalidade
│  ├─ Spam e flood não são permitidos
│  ├─ CAPS LOCK É PERMITIDO
│  ├─ Divulgação sem autorização da staff é proibida
│  └─ Memes apenas nos canais corretos
├─ CANAIS_DE_VOZ
│  ├─ Respeito é obrigatório nos voice
│  ├─ Proibido gritar ou causar poluição sonora
│  ├─ Soundboard com moderação
│  ├─ Música apenas em canal próprio
│  └─ Siga as orientações da staff
├─ X-CLÃ
│  ├─ Apenas membros autorizados participam
│  ├─ Respeito entre clãs é obrigatório
│  ├─ Provocações excessivas não são permitidas
│  └─ Fair Play sempre
├─ X-TREINO
│  ├─ Não atrapalhar treinos em andamento
│  ├─ Seguir instruções do líder ou staff
│  ├─ Comunicação clara durante o treino
│  └─ Objetivo: evolução do clã
└─ STAFF_E_PUNIÇÕES
   ├─ Respeite a staff
   ├─ Fingir ser staff = punição grave
   ├─ Punições possíveis:
   │  ├─ Advertência
   │  ├─ Mute
   │  ├─ Kick
   │  └─ Ban
